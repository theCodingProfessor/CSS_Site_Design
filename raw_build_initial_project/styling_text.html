How to Style Text on a Web Page using CSS

Applying style to text is important on a web page.

We can use CSS to change the font style, font color, the size of a font displayed, and manage how the text is displayed (in relation to other objects) all through the use of CSS.

Take for example a paragraph tag which might resemble this:

<p>This is a sample paragraph.</p>

If we want to style this paragraph to be red (the color of the text) we can apply a style to the paragraph tag as follows:

p { color: red; }

This will change all paragraphs on the page to be red.

We can apply the same concept to other page elements like headers, articles, asides and spans.

<style>
h2 { color: red; }
h3 { color: green; }
h4 { color: blue; }
h5 { color: orange; }
</style>
