Sample Content Page

The main content for each page (word describing the style technique) will be simple text surrounded by <p> (paragraph tags). The paragraph tag will need to be styled so as to avoid the need add line breaks or <br> (break tags after every paragraph.

This padding might take the form of bottom-border: 10px.

<style>
  article {
    padding-bottom: 10px;
    font-size: 1.5em; }
  article > p {
    margin-left: 1.5em; }
  article > blockquote {
    margin-left: 2.5em; }
</style>

Although the code (which sits inside the article as a blockquote also makes use of a bottom-border: 10px setting, each code tag will need to be followed by a <br/> (break tag) to ensure code lines do not run together.
